# express 3D model viewer server
## tool
- package manager: pnpm
- web app framework: express
- template engine: ejs
- 3D viewer:
    - \<model-viewer\>
    - babylon
    - javascript 3d model viewer
    - three.js

## model resource
- sketchfab

## usage
1. use git clone this repository
2. install node.js and pnpm
3. `pnpm install`
4. start server `pnpm start`  
    dev server `pnpm dev`
